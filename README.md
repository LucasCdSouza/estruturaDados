# Estrutura de Dados

Repositório com projetos e exercícios da disciplina de **Estrutura de Dados** em **Python**.

## Estrutura do repositório

- `aulas/` – Exercícios e exemplos vistos em aula.
- `DesafioClima/` – Projeto prático para manipulação de dados climáticos.  
  O desafio consiste em **ler um arquivo CSV com informações de temperatura e chuva**, processar os dados e **identificar o ano mais quente, mais frio e mais chuvoso**. É um bom exercício de **listas, dicionários e funções em Python**.
